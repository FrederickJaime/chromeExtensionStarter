# Dogs Calendar Extension

## set up
this is a plain static page with js/css/html, no real dependency, either run live server from VS Code or use Web Server for Chrome to view locally:
[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
[Web server for Chrome](https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb?hl=en)

Live server and Web Server for Chrome are optional

## testing
you can upload your current extension @ chrome://extensions/ ( testing in Chrome ),
make sure 'Developer mode' is turn on in order to upload and test locally
